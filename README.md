# Calculator

### HTML

The HTML file of this project contains a heading, created using `p` tag and a  bunch of `div` tags to help with CSS styling and adding event listners.

### CSS

The CSS in this file is primarily focused on flex-box. Miscellaneous styling are also added for margin, padding, background-color, etc.

### JS

Functions to add, subtract, multiply and divide are created and called using ooerate function. Event listener is added to all the buttons and the core logic of the program is run when buttons are clicked.
